> The goal of this project is to create a fully compatible **pong game** that will run on an [M5 Stack](https://m5stack.com/) along with 2 joysticks.

## Final goals : 

### Working pong

- 2 playable bars
- Ball
- Collisions
- Continue game on score

### Game management

- Points system
- Set / Lives system
- Start game / stop game

### Extras

- Color changes
- Sounds
